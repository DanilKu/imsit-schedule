# 🤖 Настройка Telegram бота для пользователей

## 📋 Что создано

### 1. **Вебхук бота** (`api/user_telegram_webhook.php`)
- Обработка команд `/start`, `/help`
- Инлайн-кнопки для навигации
- Расписание (сегодня/неделя с генерацией изображения)
- Просмотр заказов пользователя
- Настройки группы
- Привязка Telegram аккаунта

### 2. **Скрипт настройки** (`setup_user_telegram_webhook.php`)
- Автоматическая установка вебхука
- Проверка статуса бота
- Создание папки temp

### 3. **API привязки** (`api/link_telegram.php`)
- Привязка/отвязка Telegram ID
- Проверка статуса привязки

### 4. **Страница настроек** (`telegram_settings_user.php`)
- Интерфейс для привязки Telegram
- Инструкции для пользователей
- Статус привязки

## 🚀 Пошаговая настройка

### Шаг 1: Создание бота
1. Найдите [@BotFather](https://t.me/BotFather) в Telegram
2. Отправьте `/newbot`
3. Введите имя бота (например: "imsitID Bot")
4. Введите username (например: "imsit_bot")
5. Скопируйте токен бота

### Шаг 2: Настройка вебхука
1. Откройте `setup_user_telegram_webhook.php`
2. Замените `YOUR_USER_BOT_TOKEN_HERE` на ваш токен
3. Запустите скрипт в браузере
4. Проверьте, что вебхук установлен успешно

### Шаг 3: Настройка файлов
1. В `api/user_telegram_webhook.php` замените токен бота
2. Создайте папку `temp` в корне сайта
3. Установите права 755 на папку `temp`

### Шаг 4: Обновление ссылок
В файле `api/user_telegram_webhook.php` замените:
- `@imsit_channel` на ваш канал
- `@imsit_support` на техподдержку
- `imsit.shop` на ваш домен

## 🎯 Функции бота

### Главное меню
```
📅 Расписание    📋 Мои заказы
⚙️ Настройки     ❓ Помощь
```

### Расписание
- **Сегодня** - расписание на текущий день
- **Неделя** - генерация изображения с полным расписанием

### Заказы
- Список всех заказов пользователя
- Статусы: Ожидает, В работе, Готов, Отменен
- Даты создания

### Настройки
- Выбор группы по умолчанию
- Привязка/отвязка Telegram

## 🔧 Технические детали

### База данных
Бот использует существующие таблицы:
- `users` - для привязки Telegram ID
- `schedule` - для расписания групп
- `orders` - для заказов пользователей

### Генерация изображений
- Создается PNG изображение 800x600
- Темная тема с синими акцентами
- Показывает все пары на 1 и 2 неделю
- Автоматически удаляется после отправки

### Безопасность
- Проверка авторизации пользователя
- Валидация Telegram ID
- Защита от дублирования привязок

## 📱 Использование

### Для пользователей:
1. Зайти в настройки Telegram на сайте
2. Нажать "Привязать Telegram"
3. Найти бота @imsit_bot
4. Отправить /start
5. Следовать инструкциям

### Для администратора:
1. Мониторинг через логи
2. Управление через админку
3. Настройка уведомлений

## 🐛 Отладка

### Проверка вебхука:
```bash
curl -X POST https://api.telegram.org/bot<TOKEN>/getWebhookInfo
```

### Логи:
- Ошибки записываются в error_log
- Все входящие обновления логируются

### Тестирование:
1. Отправьте /start боту
2. Проверьте ответ
3. Протестируйте все кнопки

## 📞 Поддержка

При возникновении проблем:

2. Убедитесь, что вебхук установлен
3. Проверьте права на папку temp
4. Посмотрите логи сервера
