# Промпт для пункта: 1.2

## Заголовок

**1.2 Анализ существующих способов ведения расписания занятий**

## Промпт для нейросети

Напиши раздел отчета с заголовком **«1.2 Анализ существующих способов ведения расписания занятий»**.

Сначала обоснуй актуальность анализа, затем рассмотри основные способы ведения и публикации расписания, выдели их достоинства и недостатки, после этого покажи необходимость разработки специализированной информационной системы.

Полностью адаптируй содержание под тему **«Разработка информационной системы электронного расписания для учебного заведения»**.

Раскрой следующие способы ведения расписания:
- бумажные таблицы и стенды в учебных корпусах;
- электронные таблицы (Microsoft Excel, Google Sheets);
- модульные решения в составе ERP и электронных информационно-образовательных сред (1С, Moodle и аналоги);
- специализированные информационные системы электронного расписания;
- самостоятельно разрабатываемые web-приложения для конкретной образовательной организации.

Для каждого способа укажи:
- область применения;
- преимущества;
- ограничения и риски (ошибки ручного ввода, отсутствие централизации, сложность актуализации, ограниченный мобильный доступ).

В заключительной части обоснуй, что для дипломного проекта целесообразна разработка специализированной web-системы с централизованной базой данных, поддержкой двухнедельного цикла, разграничением прав доступа и функциями просмотра расписания по группам и преподавателям.

Не упоминай мессенджер Telegram; вместо него используй слово «мессенджер».

Ссылки в тексте оформляй в квадратных скобках по возрастанию.
