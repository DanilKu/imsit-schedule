# 🤖 Brawl Stars Telegram Bot

Telegram бот для просмотра статистики клуба в игре Brawl Stars. Полностью независимый от основного проекта.

## 📋 Возможности

- 📊 Полная статистика клуба (название, трофеи, участники)
- 👥 Список участников с трофеями (сортировка по трофеям)
- 🏆 Информация о лидерах клуба
- 📱 Удобные кнопки для быстрого доступа
- ⚡ Быстрая загрузка данных

## 🚀 Быстрый старт

### Шаг 1: Создание Telegram бота

1. Найдите [@BotFather](https://t.me/BotFather) в Telegram
2. Отправьте команду `/newbot`
3. Введите имя бота (например: "Brawl Stars Club Stats")
4. Введите username (например: "brawlstars_club_bot")
5. **Скопируйте токен бота** (выглядит как `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`)

### Шаг 2: Получение API ключа Brawl Stars

1. Перейдите на [developer.brawlstars.com](https://developer.brawlstars.com/)
2. Войдите через Supercell ID
3. Создайте новый API ключ
4. **Скопируйте API ключ** (выглядит как длинная строка символов)
5. **Важно:** Сохраните API ключ в безопасном месте, он показывается только один раз!

### Шаг 3: Получение тега клуба

1. Откройте игру Brawl Stars
2. Зайдите в свой клуб
3. Скопируйте **тег клуба** (например: `ABC123`)
4. **Важно:** Тег должен быть без символа `#`

### Шаг 4: Настройка бота

#### Вариант 1: Автоматическая настройка (рекомендуется)

1. Откройте файл `setup_brawlstars_bot.php` в редакторе
2. Замените следующие значения:
   ```php
   $botToken = 'ВАШ_ТОКЕН_БОТА';
   $brawlStarsApiKey = 'ВАШ_API_КЛЮЧ';
   $clubTag = 'ТЕГ_КЛУБА'; // без символа #
   $webhookUrl = 'https://ваш-домен.com/api/brawlstars_bot_webhook.php';
   ```
3. Откройте `setup_brawlstars_bot.php` в браузере
4. Следуйте инструкциям на экране

#### Вариант 2: Ручная настройка

1. Откройте файл `api/brawlstars_bot_webhook.php`
2. Найдите секцию `--- CONFIG ---` в начале файла
3. Замените значения:
   ```php
   $BOT_TOKEN = 'ВАШ_ТОКЕН_БОТА';
   $BRAWL_STARS_API_KEY = 'ВАШ_API_КЛЮЧ';
   $CLUB_TAG = 'ТЕГ_КЛУБА'; // без символа #
   ```
4. Установите вебхук вручную:
   ```bash
   curl -X POST "https://api.telegram.org/botВАШ_ТОКЕН/setWebhook" \
        -H "Content-Type: application/json" \
        -d '{"url":"https://ваш-домен.com/api/brawlstars_bot_webhook.php"}'
   ```

### Шаг 5: Проверка работы

1. Найдите вашего бота в Telegram (по username, который вы указали)
2. Отправьте команду `/start`
3. Нажмите кнопку "🏆 Статистика клуба"
4. Должна появиться статистика вашего клуба

## 📱 Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Начать работу с ботом |
| `/club` или `/stats` | Полная статистика клуба |
| `/short` | Краткая статистика клуба |
| `/help` | Справка по использованию |

## 🎯 Использование

### Полная статистика клуба

Отправьте команду `/club` или нажмите кнопку "🏆 Статистика клуба". Вы получите:

- 📛 Название клуба
- 🏷️ Тег клуба
- 💎 Трофеи клуба
- 📊 Минимум трофеев для вступления
- 👥 Количество участников
- 🔒 Тип клуба (открытый/закрытый)
- 📝 Описание клуба
- 👥 Список участников с трофеями (топ-20)

### Краткая статистика

Отправьте команду `/short` или нажмите кнопку "📊 Краткая статистика". Вы получите:

- 📛 Название клуба
- 💎 Трофеи клуба
- 👥 Количество участников

## 🔧 Настройка через переменные окружения (опционально)

Вместо хранения токенов в коде, вы можете использовать переменные окружения:

```bash
export BRAWLSTARS_BOT_TOKEN="ваш_токен_бота"
export BRAWL_STARS_API_KEY="ваш_api_ключ"
export BRAWL_STARS_CLUB_TAG="тег_клуба"
```

Бот автоматически использует эти переменные, если они установлены.

## ⚠️ Важные замечания

1. **Безопасность:**
   - Никогда не публикуйте токен бота и API ключ в открытом доступе
   - Не коммитьте файлы с токенами в публичные репозитории
   - Используйте переменные окружения для продакшена

2. **Лимиты API:**
   - Brawl Stars API имеет лимиты на количество запросов
   - Не злоупотребляйте частыми запросами
   - Рекомендуется кэшировать данные на несколько минут

3. **Тег клуба:**
   - Тег должен быть указан **без символа #**
   - Тег чувствителен к регистру (ABC123 ≠ abc123)
   - Убедитесь, что скопировали тег правильно

## 🐛 Решение проблем

### Бот не отвечает

1. Проверьте, что вебхук установлен правильно:
   ```bash
   curl "https://api.telegram.org/botВАШ_ТОКЕН/getWebhookInfo"
   ```
2. Проверьте логи сервера на наличие ошибок
3. Убедитесь, что URL вебхука доступен из интернета

### Ошибка "Не удалось получить данные о клубе"

1. Проверьте правильность тега клуба
2. Убедитесь, что API ключ действителен
3. Проверьте, что клуб существует и доступен
4. Проверьте лимиты API (возможно, превышен лимит запросов)

### Ошибка "Ошибка конфигурации"

1. Убедитесь, что все токены установлены в файле `brawlstars_bot_webhook.php`
2. Проверьте правильность формата токенов
3. Убедитесь, что нет лишних пробелов в значениях
